📌 Descrição

Este programa em C# permite que o usuário digite quatro nomes e, em seguida, exibe esses nomes na ordem invertida.

⚙️ Funcionalidades
Recebe 4 nomes digitados pelo usuário
Armazena os nomes em variáveis
Inverte a ordem dos nomes usando uma variável auxiliar
Exibe os nomes já invertidos no console
💻 Como funciona
O programa solicita ao usuário que digite quatro nomes
Cada nome é armazenado em uma variável (nome1, nome2, nome3, nome4)
A inversão acontece em duas etapas:
Troca nome1 com nome4
Troca nome2 com nome3
O resultado final é a sequência invertida dos nomes digitados
Os nomes são exibidos na tela
🔄 Lógica da inversão

Exemplo:

Entrada:

nome1 = Ana
nome2 = Bruno
nome3 = Carlos
nome4 = Daniel

Saída:

Daniel
Carlos
Bruno
Ana
🧠 Tecnologias utilizadas
Linguagem: C#
Entrada: Console.ReadLine()
Saída: Console.WriteLine()
▶️ Como executar
Abra o Visual Studio ou VS Code
Crie um projeto do tipo Console Application
Cole o código no arquivo principal
Execute o programa
Digite os nomes quando solicitado
⚠️ Observações
O programa não faz validação de entrada (aceita qualquer texto)
A quantidade de nomes é fixa (4 nomes)
